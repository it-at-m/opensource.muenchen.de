OpenJDK is the official free implementation of the Java Platform, Standard Edition (Java SE).

---

OpenJDK is the open source reference implementation of the Java specification.
Only OpenJDK distributions are used for LHM - in the JRE or JDK version suitable for the intended use.

The LHM standard is [Red Hat OpenJDK](https://developers.redhat.com/products/openjdk/download).
In addition, Adoptium OpenJDK and OpenWebStart are available as alternatives, which are functionally equivalent solutions to Java.
