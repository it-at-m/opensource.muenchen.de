The __Agile Test Automation Framework__ (ATAF) is a [Java](java) 21 framework for Cucumber-based BDD tests and traditional TestNG/JUnit test suites, with optional Jira/Xray integration.

---

It is designed for agile projects: fast setup, maintainable test automation, and integration into modern development workflows.
In addition to browser-based and API testing, it provides hooks for managing test executions in Jira and Xray.
